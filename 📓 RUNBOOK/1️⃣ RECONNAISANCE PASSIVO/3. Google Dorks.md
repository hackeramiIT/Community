### Introduzione
Il Google Dorking (o Google Hacking) è una tecnica di OSINT che utilizza operatori di ricerca avanzati per identificare informazioni sensibili esposte accidentalmente su Internet. Consiste nell'uso di query specifiche per:

- Trovare dati riservati indicizzati da Google
- Identificare vulnerabilità e configurazioni errate
- Scoprire asset non protetti
- Mappare l'infrastruttura target

---
### Operatori

| **Operatore**                                      | **Spiegazione**                                      |
| -------------------------------------------------- | :--------------------------------------------------- |
| `site:`                                            | Limita la ricerca a un dominio o sottodominio        |
| `filetype:` / `ext:`                               | Cerca file con una certa estensione (es. PDF, xls)   |
| `inurl:` / `allinurl:`                             | Cerca parole nell’URL                                |
| `intitle:` / `allintitle:`                         | Cerca parole nel titolo della pagina (tag `<title>`) |
| `intext:` / `allintext:`                           | Cerca parole nel corpo della pagina                  |
| `cache:`                                           | Mostra la versione in cache di Google di una pagina  |
| `related:`                                         | Trova siti simili a un dominio                       |
| `link:`                                            | Trova pagine che linkano a un URL (limitato)         |
| `before:` / `after:`                               | Filtra risultati per data (quando disponibile)       |
| `inanchor:`                                        | Cerca testo usato come anchor text nei link          |
| `AROUND(n)`                                        | Trova termini a una distanza n (prossimità)          |
| `site:`, combinazioni logiche (AND/OR, virgolette) | Permettono ricerche più mirate combinando operatori  |
